# Changelog

## 1.0.0
- First public release.
- Install from Forge search, mod links, mod-list links, or local .zip / .7z files.
- Unified install queue with progress, abort/skip, and dependency auto-resolution.
- Manage tab: single list with client/server/both tags, enable/disable, update
  checks with changelogs, multi-select uninstall, search, filters, disk usage.
- SPT 4.0 aware install placement; metadata + icon caching; portable single exe.
